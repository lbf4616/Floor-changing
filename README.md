# Floor-changing

[[ENGLISH]](README_en.md)  [[中文]](README.md)  

AI智能换地板，换瓷砖，换墙漆，已为多个地板品牌商提供服务，欢迎联系。

◆ 使用神经网络分割出地板区域。  
◆ 使用3D算法计算透视角度。  
◆ 使用传统图像算法进行光影融合。
